Clawbolt can generate professional PDF estimates from a natural language description. Describe the job, and Clawbolt creates a formatted PDF with line items, totals, and your contractor info.

## How it works

1. **Describe the job**: Tell Clawbolt about the work, materials, and pricing
2. **Clawbolt builds the estimate**: The agent extracts line items, quantities, and unit prices
3. **PDF generated**: A professional estimate is created using ReportLab
4. **Delivered instantly**: The PDF is sent back in the Telegram chat and stored in your cloud storage

## Example conversation

```
You: I need an estimate for the Johnson kitchen remodel.
     Demo old cabinets $800, install new cabinets $2400,
     countertop install $1800, plumbing hookup $600

Clawbolt: Here's your estimate for the Johnson kitchen remodel:

          EST-0042 | Johnson Kitchen Remodel
          - Demo old cabinets: $800.00
          - Install new cabinets: $2,400.00
          - Countertop installation: $1,800.00
          - Plumbing hookup: $600.00
          Subtotal: $5,600.00

          [PDF attached]
```

## Estimate lifecycle

Estimates move through these statuses:

| Status | Description |
|--------|-------------|
| **Draft** | Just created, ready for review |
| **Sent** | Forwarded to the client |
| **Accepted** | Client approved the estimate |
| **Rejected** | Client declined the estimate |

## What's included in the PDF

- Your contractor name, trade, and contact info (from your profile)
- Client name and address (if known)
- Itemized table with descriptions, quantities, and unit prices
- Subtotal and total calculations
- Custom terms (if provided)
- Unique estimate number (EST-0001 format)

## Tips

- Clawbolt uses your saved rates from [memory](../memory/) when available
- Include client details and Clawbolt will add them to the estimate header
- Estimates are automatically stored in your [cloud storage](../../deployment/storage/) under an Estimates folder
