---
title: File Cataloging
description: Automatically organize photos and documents in cloud storage.
---

Clawbolt can automatically catalog job photos, voice notes, and documents to a cloud storage backend. Files are organized by client and type.

## Supported storage providers

| Provider | Setup |
|----------|-------|
| **Local** | Works out of the box. Files saved to `data/storage/` on disk |
| **Dropbox** | Requires API credentials. See [Storage Providers](../../deployment/storage/) |
| **Google Drive** | Requires OAuth credentials. See [Storage Providers](../../deployment/storage/) |

## File organization

Files are organized into folders by type:

```
Job Photos/
├── 2026-02-28/
│   ├── site-front.jpg
│   └── site-back.jpg
└── 2026-03-01/
    └── kitchen-demo.jpg
```

## How it works

When you send media, the agent uses the `upload_to_storage` and `organize_file` tools to:

1. Upload the file to your configured storage backend
2. Organize it into the appropriate folder structure
3. Generate a shareable link (for Dropbox/Google Drive)
4. Store the storage URL in the user's media manifest for future reference

## Configuration

Set the `STORAGE_PROVIDER` environment variable to choose your backend. See [Storage Providers](../../deployment/storage/) for detailed setup instructions for each provider.
